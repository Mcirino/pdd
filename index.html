Welcome to PaleoDeepDive

The PaleoDeepDive statistical machine reading and learning system was developed to extract paleontological data from scientific literature, ie. dark data. DeepDive is the base of PaleoDeepDive. DeepDive is a trained system, which means it learns and gets better with repeated iterations and more data. PaleoDeepDive is a variant of DeepDive specializing in extracting data from paleontological literature, and outputting useful fossil data, specifically fossil size data.

Using PaleoDeepDive

PaleoDeepDive requires the installaiton of the 0.8.0-alpha version of DeepDive, available here. A PaleoDeepDive walkthrough is available here.
